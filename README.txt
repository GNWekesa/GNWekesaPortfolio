## Development Notes
1. **Dependencies**: Requires active internet connection for CDN resources
2. **Customization**: Theme colors controlled through CSS variables
3. **Extension Points**: 
   - Formspree integration for contact form
   - GitHub API for dynamic project loading
   - Intersection Observer for scroll animations
4. **Image Optimization**: 
   - Recommended WebP format
   - 300-400px width for carousel images
   - 1200px max-width for hero images

## Usage Instructions
1. Clone repository
2. Update content in HTML sections
3. Replace placeholder images
4. Deploy as static site
5. Monitor form submissions (backend required)
